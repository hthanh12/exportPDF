# koajs-api
koajs + postgres

## Quickstart

### Install dependencies:
```$ npm i```

### Create file .env:

```$ touch .env```

Copy to .env
```
NODE_ENV='development'
PORT=3000
```

### Start the server:
```$ npm start```

View the website at: http://localhost:3000
